# WEB Test

## Dependencies

Git
NodeJs 8 or Greater
NPM

## Clone, Build & Run

```
git clone https://github.com/MartinMelo/Hotels
cd Hotels
npm install
cd public
npm install
ng build
cd ..
npm start
```

Runs at port 80
